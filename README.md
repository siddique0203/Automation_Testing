**AgriConnect Contact Form Automation**

Automated Contact Form Submission Using Selenium WebDriver (Python)

This repository contains a Selenium automation script to test the Contact form of the AgriConnect
 platform. The script opens the registration page, navigates to the Contact section, fills in the required details, and submits the form.

--Features

-Open AgriConnect registration page in Chrome browser

-Navigate to Contact section

-Fill out Name, Email, Phone, Subject, and Message fields

-Submit the contact form automatically

-Validate form submission
